# Prompts de Vídeo 16:9 para os 24 Exercícios no Veo 2 Fast

Este arquivo foi refeito para um fluxo principal com `frame inicial + frame final`, priorizando `segurança biomecânica`, `execução controlada`, `amplitude conservadora` e `leitura técnica clara` acima de dramatização visual.

## Como usar

- Gere o `frame inicial` com o prompt de imagem correspondente.
- Gere o `frame final` com o prompt de imagem correspondente.
- No `Flow`, selecione `Veo 2 - Fast`, `16:9`, `Frames to Video: First + last`.
- Anexe o frame inicial e o frame final.
- Cole `somente` o `Prompt de vídeo` do exercício.

## Regras globais

- Todos os prompts assumem `8 segundos`.
- Todos os prompts assumem `uma cena`, `uma câmera`, `uma transição controlada`.
- O objetivo não é performance atlética; é `demonstração segura e realista`.
- A personagem deve parecer uma mulher adulta iniciante, não atleta profissional.
- O movimento deve parecer `deliberado`, `controlado`, `sem impulso`, `sem dor`, `sem exagero`.
- O enquadramento deve manter `corpo inteiro`, `apoio`, `mãos`, `pés` e `contato com o chão` visíveis quando relevantes.

## Convenção de congruência

- Em exercícios unilaterais, use sempre o `mesmo lado` no frame inicial, no prompt de vídeo e no frame final.
- Em vistas `3/4 lateral` ou `lateral`, preserve o `mesmo lado do corpo` voltado para a câmera do início ao fim.
- O vídeo não deve `espelhar`, `girar`, `trocar de lado` ou `orbitar` ao redor da personagem entre os frames.
- Quando um exercício exigir um lado específico, este arquivo já fixa esse lado no próprio prompt.
