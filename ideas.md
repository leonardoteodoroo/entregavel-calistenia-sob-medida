## Ideia A — Editorial Wellness Minimalista

**Design Movement:** Editorial wellness / Scandinavian minimalism com toque feminino

**Core Principles:**
1. Espaço em branco como elemento de design ativo — respiro visual em cada página
2. Hierarquia tipográfica forte com contraste de peso (serif elegante + sans legível)
3. Paleta restrita e sofisticada: ivory, taupe, rose escuro, teal suave
4. Conteúdo estruturado como revista premium, não como apostila

**Color Philosophy:**
- Ivory (#F9F6F0) como fundo base — quente, sofisticado, não frio
- Taupe (#B5A99A) para elementos secundários e bordas sutis
- Rose escuro (#8B4A52) como cor de destaque — feminino sem ser infantil
- Teal suave (#5B8A8B) para variação e equilíbrio visual
- Texto principal em charcoal escuro (#2C2C2C) para máxima legibilidade

**Layout Paradigm:**
- Páginas tipo A4 com margens generosas
- Títulos alinhados à esquerda com linha decorativa fina
- Blocos de exercício em cards com fundo levemente diferenciado
- Numeração de página discreta no rodapé
- Divisores orgânicos entre seções

**Signature Elements:**
1. Linha horizontal fina em rose escuro como separador de seção
2. Inicial decorativa grande no início de textos de boas-vindas
3. Ícones minimalistas de linha fina para categorias de exercício

**Interaction Philosophy:**
- Navegação por sidebar discreta com indicador de progresso
- Transições suaves entre páginas (fade)
- Checklist interativo para marcar dias concluídos

**Animation:**
- Fade-in suave ao entrar em cada seção (300ms ease-out)
- Underline animado em links de navegação
- Checkbox com animação de check suave

**Typography System:**
- Títulos: Playfair Display (serif elegante) — peso 400/700
- Corpo: DM Sans (sans humanista) — peso 300/400
- Destaques: DM Sans Italic ou Playfair Display Italic
- Tamanho base: 16px, escala modular 1.25

## Ideia B — Wellness Orgânico com Textura

**Design Movement:** Organic wellness / Natural editorial com elementos texturizados

**Core Principles:**
1. Textura sutil como fundo (papel, grain) para sensação premium tátil
2. Formas orgânicas e arredondadas em destaques
3. Paleta terrosa e natural com acento floral
4. Layout assimétrico com âncoras visuais fortes

**Color Philosophy:**
- Creme texturizado (#F5F0E8) como base
- Terracota suave (#C4856A) como acento quente
- Sage verde (#7A9E7E) para equilíbrio natural
- Nude profundo (#9E8070) para elementos secundários

**Layout Paradigm:**
- Layout de duas colunas em páginas de treino
- Coluna esquerda: informações do treino; coluna direita: exercícios
- Elementos decorativos botânicos sutis nos cantos
- Blocos de texto com fundo levemente colorido

**Signature Elements:**
1. Formas orgânicas (blob) como fundo de destaque
2. Elementos botânicos lineares decorativos
3. Badges arredondados para versões de intensidade

**Interaction Philosophy:**
- Scroll suave com parallax leve em elementos decorativos
- Hover com elevação sutil em cards de exercício
- Progresso visual com barra orgânica

**Animation:**
- Slide-up ao entrar em seções (400ms ease-out)
- Elementos decorativos com rotação lenta (20s loop)
- Transição de página com wipe horizontal

**Typography System:**
- Títulos: Cormorant Garamond (serif clássico) — peso 300/600
- Corpo: Nunito (sans amigável) — peso 300/400
- Destaques: Cormorant Garamond Italic

## Ideia C — Premium Wellness com Assimetria Ousada

**Design Movement:** Premium wellness editorial com influência de moda

**Core Principles:**
1. Assimetria intencional — elementos fora do grid criam tensão visual positiva
2. Tipografia como elemento gráfico — títulos grandes e expressivos
3. Contraste entre áreas densas e áreas de respiro
4. Sofisticação através da restrição — menos é mais

**Color Philosophy:**
- Branco quente (#FDFAF5) como base
- Preto suave (#1A1A1A) para tipografia de impacto
- Blush (#D4A5A5) como único acento colorido
- Linha dourada (#C9A96E) para elementos de prestígio

**Layout Paradigm:**
- Títulos de página em tamanho display (80-100px) cortando o layout
- Conteúdo em coluna estreita com muito espaço lateral
- Numeração de página em destaque tipográfico
- Blocos de exercício em lista limpa sem cards

**Signature Elements:**
1. Número de página em tipografia display como elemento decorativo
2. Linha vertical fina como separador de coluna
3. Letras iniciais decorativas em blush

**Interaction Philosophy:**
- Navegação por índice lateral fixo
- Transição de página com slide vertical
- Hover sutil em elementos interativos

**Animation:**
- Reveal de texto linha por linha (stagger 50ms)
- Fade-in de elementos com delay progressivo
- Cursor personalizado em áreas interativas

**Typography System:**
- Títulos: Bodoni Moda (serif display) — peso 400/700
- Corpo: Jost (sans geométrico) — peso 300/400
- Destaques: Bodoni Moda Italic
</text>

## Decisão Final: Ideia A — Editorial Wellness Minimalista

Escolhida por melhor equilibrar elegância, legibilidade e acessibilidade para o público-alvo (mulheres iniciantes com rotina corrida). A combinação Playfair Display + DM Sans entrega sofisticação sem intimidar. A paleta ivory/taupe/rose/teal é exatamente a solicitada pelo cliente.
